Detailed Description

Library for manipulating an ordered vector of integers with fixed interval.

This library implements a count vector which is a vector based on the count sort algorithm.

The structure defined in this library consists in a vector whose positions stores the number of repetitions of the number in the vector. For instance, if a vector v have 47 repetitions of the number 3 then we have v[3] = 47.
