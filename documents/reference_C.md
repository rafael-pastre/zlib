# Reference for C programming

## Error Handling
* assert() at <assert.h>

## Interruption
* signal() at <signal.h>

## Variable number of function arguments
* va_list at <stdarg.h>

## Extra types <stddef.h>
* ptrdiff_t: signed number that results from pointer subtraction
* size_t: result of sizeof()
* wchar_t: wide character (?)
